Normal jQuery .hover method

Hint: Move pointer in and out of box rapidly.

$('#normal').hover(function(e) {
  $(this).animate({'height': '200px'});
}, function() {
  $(this).animate({'height': '100px'});
});

Enhanced .hover method with delay

$('#delay').hover(function(e) {
  $(this).animate({'height': '200px'});
}, function() {
  $(this).animate({'height': '100px'});
}, 250);
